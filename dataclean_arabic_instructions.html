دليل استخدام DataClean Pro بالعربي

دليل الاستخدام
DataClean Pro — تحويل ملفات CSV إلى Excel بشكل تلقائي
الملفات المرفقة
التطبيق الرئيسي
app.py
الواجهة البصرية — ترفع فيها ملفات CSV وتنزّل Excel بضغطة زر
المحرك الداخلي
clean_csv.py
الكود الذي يشتغل خلف التطبيق — لا تحتاج تفتحه أو تعدّل فيه
مطلوب أولاً
requirements.txt
قائمة بالمكتبات اللي يحتاجها البرنامج — يُنصّب مرة واحدة فقط
خطوات التثبيت — مرة واحدة فقط
١
تنزيل Python
افتح الموقع python.org/downloads ونزّل أحدث نسخة.
أثناء التثبيت، تأكد تضع علامة ✓ على "Add Python to PATH" قبل ما تضغط Install.
٢
فتح Terminal أو Command Prompt
على Windows: اضغط Win + R واكتب cmd ثم Enter.
على Mac: افتح Spotlight وابحث عن Terminal.
٣
الانتقال إلى مجلد البرنامج
اكتب الأمر التالي واستبدل المسار بمكان مجلد DataCleanPro عندك:
cd C:\Users\اسمك\Desktop\DataCleanPro
٤
تنصيب المكتبات — مرة واحدة فقط
اكتب هذا الأمر واضغط Enter وانتظر حتى ينتهي:
pip install -r requirements.txt
هذا كل شي، ما تحتاج تكرره بعد كذا.
٥
تشغيل التطبيق
اكتب هذا الأمر واضغط Enter:
streamlit run app.py
سيفتح المتصفح تلقائياً على التطبيق. هذا الأمر تكتبه كل مرة تريد تشغّل البرنامج.
كيف تستخدم التطبيق
ارفع الملف
اسحب ملف CSV أو أكثر وأفلته في منطقة الرفع
اختر الإعدادات
من القائمة الجانبية اختر كيف تتعامل مع القيم الفارغة
راجع النتائج
يظهر تقرير بكل ما تم تنظيفه وعدد الصفوف المحذوفة
نزّل Excel
اضغط زر التنزيل واحصل على ملف Excel نظيف وجاهز
ملاحظات مهمة
لو البرنامج قال "no rows left" — غيّر الاستراتيجية في الشريط الجانبي إلى "Fill with mean / median" بدل حذف الصفوف.
البرنامج يدعم ملفات CSV بأي ترميز أو فاصل (فاصلة، فاصلة منقوطة، tab) — يكتشف كل شي تلقائي.
لا تغلق نافذة Terminal أثناء استخدام التطبيق — لو أغلقتها سيتوقف البرنامج. لإيقافه بشكل صحيح اضغط Ctrl + C في Terminal.
